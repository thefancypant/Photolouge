Installation Process

Setup Machine for python development
  1. Install python package manager. https://pip.pypa.io/en/stable/installing/

  2. Install tool libraries:

    pip install ipython ipdb virtualenv virtualenvwrapper

Note: You may be required to use sudo

Starting Django Project
  1. Goto workspance directory
  2. mkdir project_dir and cd to it.

  3. create virtualenv within the project dir

    virtualenv .venv

  4. activate the virtualenv: source .venv/bin/activate

  5. Install libs:

    pip install ipython ipdb django

  6. Start Django Project: 

    django-admin startproject project_name
Configuring PyCharm
  1. Open the Projectdir in pycharm
  2. From File > Settings >Project Interpreter, set the python interpreter to projectdir/.venv/bin/python
  3. Right Click on projectdir/project_name folder and Mark Dir as Root.
